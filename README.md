# 米哈云游启动器

**米哈云游启动器** 是首个米家云游戏2合1的启动器，由开发者 JackTao 创作。这款启动器使用 Python 语言编写，并遵循 GPL3.0 协议开源。
## 代码运行方法
- 本软件使用Python语言，作者已经编译好Windows平台 Linux，MacOS平台可以自行编译

## 主要功能

- 提供简洁的用户界面，方便用户选择要启动的云游戏。
- 支持崩坏·星穹铁道和原神两款米家云游戏。
